Brophy Immersion Trip · 2026
Peru — International Exchange & Immersion
Lima · Cusco · Arequipa

This year I'll be traveling to Peru as part of Brophy's international immersion program, a two-way exchange where Brophy students spend time in Peruvian communities and Peruvian students visit Brophy the following school year. The program is designed not as tourism but as genuine encounter: We'll be building churches, houses, and a lot of other projects, and I'll be living with my Peru buddy, Braulio, just like how he lived with me for two weeks at the start of the year.

I want to see how different life is for underserved individuals and not only help them physically like we are doing but in the future I would like to use machine learning to help them get better access to food and information.

Dopamine Jailbreak — iPhone 7, iOS 15.8.3
iOS security kernel exploits ARM64
Deployed a rootless jailbreak on iOS 15.8.3 using the Dopamine exploit chain, which leverages kernel vulnerabilities to bypass Apple's code-signing enforcement. Bootstrapped the environment with TrollStore for persistence, installed Sileo as a package manager, and tested a suite of system-level tweaks — exploring how user-space processes interact with the sandboxed iOS kernel.
Luma3DS CFW — New 3DS XL
ARM11/ARM9 custom firmware embedded
Installed Luma3DS custom firmware on a New 3DS XL by exploiting the system's ARM9 bootrom to gain full hardware access below the OS level. Set up CIA installation for homebrew and retail titles, configured custom system themes, and explored the dual-core ARM architecture — ARM11 for applications, ARM9 for the privileged security processor.
PS3 Slim — Custom Firmware via USB
Cell BE architecture hypervisor bypass CFW
Flashed a PS3 Slim to custom firmware by exploiting a vulnerability in Sony's software update chain via USB. Gained hypervisor-level access to the Cell Broadband Engine, enabling unsigned code execution. Installed RetroArch and additional emulators, effectively repurposing the Cell BE's seven SPE cores for multi-system emulation workloads.
Nintendo Switch RCM Jig — Attempted CFW
Tegra X1 RCM exploit hardware mod
Fabricated a custom RCM jig from aluminum foil and packaging to short pins 1 and 10 in the right Joy-Con rail, triggering the Tegra X1's Recovery Mode — a bootrom-level vulnerability that cannot be patched via software update. After an hour of work, discovered the unit was a hardware-patched V2 model with the fixed Tegra revision. Documented for future use on an unpatched unit.

DS Lite — R4 Flashcard & NES Emulation
ARM7/ARM9 flashcart emulation
Configured an R4 flashcard on a DS Lite to run homebrew NDS ROMs by exploiting the DS's lack of code-signing verification on slot-1 cartridges. Tested NES emulation via nesDS, benchmarking performance against the ARM7's 33MHz clock — and confirmed that the DS Lite's relatively underpowered ARM7 co-processor makes for a poor NES emulation host. A useful lesson in hardware constraints.
Pi-hole — Per-Device DNS Filtering on Raspberry Pi
Raspberry Pi DNS networking Linux
Deployed Pi-hole on a Raspberry Pi as a local DNS sinkhole, configuring each device on the network with a static DNS entry pointing to the Pi rather than the router's default resolver. This intercepts DNS queries at the network layer before they reach upstream servers, allowing per-device filtering rules — blocking ad-serving and tracking domains for specific devices while leaving others unaffected. Explored how DNS resolution works end-to-end and how most ad infrastructure relies on predictable domain patterns that make sinkholing highly effective.
